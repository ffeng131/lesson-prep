# 集体备课 H5 端项目

> 一个基于原生 HTML/CSS/JavaScript 开发的集体备课移动端应用原型

## 📁 项目结构

```
./
├── index.html          # 首页（备课组选择、目录树）
├── detail.html         # 详情页（资源管理）
├── styles.css          # 全局样式
├── data.js             # Mock 数据
├── app.js              # 首页交互逻辑
├── detail.js           # 详情页交互逻辑
├── PRD.md              # 产品需求文档
└── README.md           # 项目说明文档（本文件）
```

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 下载或克隆项目到本地
2. 直接用浏览器打开 `index.html` 文件
3. 即可预览项目

### 方式二：本地服务器（推荐用于移动端预览）

```bash
# 进入项目目录
cd /path/to/project

# 启动本地服务器
python3 -m http.server 8080

# 访问地址
# 本地: http://localhost:8080/index.html
# 局域网: http://你的IP:8080/index.html
```

### 方式三：使用 Node.js

```bash
# 安装 http-server（如果未安装）
npm install -g http-server

# 启动服务器
http-server -p 8080

# 访问地址
# http://localhost:8080/index.html
```

## 📱 功能特性

### 首页功能
- ✅ 备课组名称显示
- ✅ 备课组切换（底部弹窗）
- ✅ 个备进度展示
- ✅ 教学内容目录树（多级展开）
- ✅ 状态标识（集/个/思）

### 详情页功能
- ✅ 集备资源 / 个备资源 Tab 切换
- ✅ 主备人信息展示
- ✅ 资源类型状态（教案、课件）
- ✅ 资源卡片列表
- ✅ 资源操作（编辑、添加为个备、修改类型、重命名、下载、删除）
- ✅ 文件上传（支持多类型）
- ✅ 教师资源切换（横向滚动）
- ✅ 完成集备按钮（带确认弹窗）

## 🎨 设计特点

- 📐 **响应式设计**：适配各种手机屏幕尺寸
- 🎯 **移动端优化**：触摸友好，滑动流畅
- 🎨 **现代UI**：简洁美观的界面设计
- ⚡ **轻量无依赖**：原生 JS，无需框架
